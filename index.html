→ OL - UL - LI

Elemento HTML de lista:

O Elemento HTML <ol> (ou Elemento HTML de lista ordenada) representa uma lista de itens ordenados.

O elemento HTML <ul> (ou elemento HTML de Lista desordenada) representa uma lista de itens sem ordem rígida.

O elemento HTML <li> (ou a Lista dos Itens de um elemento HTML) é usado para representar um item que faz parte de uma lista.

→ STRONG - I - U - B

O elemento HTML <strong> dá ao texto uma forte importância, e é tipicamente mostrado em negrito.

O elemento HTML <i> é usado para transmitir um significado tradicionalmente transmitido por itálico.

O elemento HTML <b> é usado para transmitir um significado tradicionalmente transmitido por negrito.

O elemento HTML <u> é usado para transmitir um significado tradicionalmente transmitido pelo sublinhado.

  • <strong> texto
  • <i> texto
  • <u> texto
  • <b> texto

→ A - ABBR - MARK - SMALL - HR

O Elemento HTML <a> (ou elemento âncora), com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL.

O Elemento HTML <abbr> representa uma abreviação e opcionalmente fornece uma descrição completa para ela.

O Elemento HMTL <mark> representa um trecho de destaque em um texto, por exemplo, uma sequência de texto marcado como referência, devido à sua relevância em um contexto particular.

O Elemento HTML <small> representa comentários paralelos e letras pequenas, como direitos autorais e texto jurídico, independentemente de sua apresentação com estilo.

O Elemento HTML <hr> representa uma ruptura temática entre os elementos no nível do parágrafo: por exemplo, uma mudança de cenário em uma história ou uma mudança de tópico em uma seção.

→ SUP - SUB

O Elemento HTML <sup> especifica o texto em linha que deve ser exibido como sobrescrito por apenas razões tipográficas. Os sobrescritos geralmente são renderizados com uma linha de base elevada usando texto menor.

O Elemento JHTML <sub> especifica o texto em linha que deve ser exibido como subscrito por apenas razões tipográficas. Os subscritos geralmente são renderizados com uma linha de base reduzida usando texto menor.

  • <sup> a2 + b2 = c2
  • <sub> C8H10N4O2
